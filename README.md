# Portfolio — Refonte (GitHub Pages)

## Déploiement GitHub Pages

1. Mets ces fichiers à la racine du repo GitHub Pages (branche `main` ou `gh-pages`).
2. Settings → Pages → Build and deployment → Source: `Deploy from a branch`.
3. Branch: `main` + folder `/ (root)`.
4. Attends 1-2 minutes.

## Structure

- `index.html` : page principale
- `css/main.css` : styles (variables + responsive)
- `js/main.js` : interactions (nav, reveal, theme, back-to-top, secret)
- `secret.html` : page "accès secret"
- `projet/certif/certifications.html` : page certifications
- `projet/timeline/index.html` : page progress
- `assets/` : reprendre les assets existants (photo, pdf, svg, qr, badges)
